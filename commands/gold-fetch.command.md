# 采集黄金数据

执行黄金数据采集脚本，获取最新市场数据并保存到 `data/today_data.json`。

## 执行逻辑

1. **检查Python环境**
   ```bash
   python --version  # 要求 >= 3.8
   ```

2. **执行采集脚本**
   ```bash
   python data/fetch_gold.py [--full] [--date YYYYMMDD]
   ```

3. **采集内容**
   - **最小采集**（默认）：518880 ETF、AU9999、SHFE黄金期货
   - **全量采集**（--full）：额外采集伦敦金、WTI原油、美元指数等

4. **输出结果**
   - 数据保存至 `data/today_data.json`
   - 控制台显示采集状态和季节性周期信息
   - 自动清理30天前的历史报告

## 参数说明

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `--full` | boolean | false | 执行全量数据采集 |
| `--date` | string | 今天 | 指定采集日期（YYYYMMDD） |

## 使用示例

```bash
# 最小采集（日常使用）
/gold-fetch

# 全量采集（周一深度报告前）
/gold-fetch --full

# 采集指定日期数据
/gold-fetch --date 20260315
```

## 故障排除

| 问题 | 解决方案 |
|------|---------|
| API调用失败 | 自动降级到缓存数据，不影响主流程 |
| 数据为空 | 检查网络连接，或尝试指定历史日期 |
| Python报错 | 确保Python >= 3.8，依赖包已安装 |

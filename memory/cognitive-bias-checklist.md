# 黄金分析认知偏差检查清单

> **适用场景**: 黄金深度分析前自检  
> **使用方式**: 每次执行 `/gold-weekly-deep` 或深度分析前完成此清单  
> **目标**: 识别并纠正认知偏差，提高分析客观性

---

## 📋 快速检查表

在正式开始黄金分析前，请逐一检查以下项目：

### 1️⃣ 确认偏误检查 (Confirmation Bias)

**问题清单**:
- [ ] 我是否已经形成了看涨/看跌的预设观点？
- [ ] 我是否只关注支持我观点的数据（如只看利多消息）？
- [ ] 我是否主动寻找了反方证据？
- [ ] 我是否考虑过"如果我看错了，市场会怎么走"？

**黄金分析场景**:
```
❌ 错误示例:
"我认为黄金会涨，所以只关注美联储降息预期，忽视美元走强信号"

✅ 正确做法:
"我倾向于看涨，但必须列出3个看跌理由：
 1. 美元指数突破105
 2. 实际利率上升
 3. 风险资产偏好回升"
```

**纠正措施**:
1. 强制列出至少3个反方观点
2. 为每个观点寻找数据支持
3. 计算看多/看空证据的权重对比

---

### 2️⃣ 锚定效应检查 (Anchoring Bias)

**问题清单**:
- [ ] 我是否被"关键价位"过度锚定（如$3000、$2800）？
- [ ] 我是否被买入成本价影响判断？
- [ ] 我是否被近期价格（上周收盘价）限制了思维？
- [ ] 我是否忽视了价格已经发生重大变化？

**黄金分析场景**:
```
❌ 错误示例:
"我在$2900买入的，现在$2850，只要回到$2900我就卖出"
（被成本价锚定，忽视当前市场逻辑已变）

✅ 正确做法:
"当前价格$2850，基于当前宏观环境，合理估值区间是$2800-$2900
 买入成本 irrelevant，只关注当前价值"
```

**纠正措施**:
1. 明确写出"如果不考虑历史价格，当前该如何判断"
2. 使用多时间框架分析（日线/周线/月线）
3. 关注价格变化率而非绝对价格

---

### 3️⃣ 过度自信检查 (Overconfidence Bias)

**问题清单**:
- [ ] 我的预测是否过于精确（如"下周会涨到$2950"）？
- [ ] 我是否为预测错误预留了足够空间？
- [ ] 我是否低估了不确定性？
- [ ] 我是否忽视了"我不知道"的情况？

**黄金分析场景**:
```
❌ 错误示例:
"根据我的分析，黄金下周一定会突破$2900，目标$2950"
（过度精确，忽视不确定性）

✅ 正确做法:
"基于当前信息，黄金下周有60%概率在$2850-$2920区间波动
 突破$2920的概率30%，跌破$2850的概率30%
 如果FOMC释放鹰派信号，区间下移至$2800-$2880"
```

**纠正措施**:
1. 使用概率区间而非点估计
2. 明确标注置信度（如"70%置信度"）
3. 列出"如果预测错误，最可能的原因是什么"

---

### 4️⃣ 损失厌恶检查 (Loss Aversion)

**问题清单**:
- [ ] 我是否因为不想承认错误而继续持有亏损头寸？
- [ ] 我是否过早获利了结（害怕利润回吐）？
- [ ] 我是否对亏损头寸投入过多关注，忽视其他机会？
- [ ] 我是否因为"已经亏了很多"而拒绝止损？

**黄金分析场景**:
```
❌ 错误示例:
"黄金已经跌了5%，我再等等，说不定会反弹"
（损失厌恶导致拒绝止损）

✅ 正确做法:
"当前亏损5%，但触发了我预设的止损条件（跌破$2800）
 根据交易计划，应该执行止损，而非期待反弹
 沉没成本 irrelevant"
```

**纠正措施**:
1. 严格执行预设的交易计划
2. 定期评估"如果今天空仓，我还会买入吗"
3. 使用机械止损，减少情绪干扰

---

### 5️⃣ 从众心理检查 (Herd Mentality)

**问题清单**:
- [ ] 我的观点是否只是市场共识的重复？
- [ ] 我是否因为"大家都看多"而看多？
- [ ] 我是否有独立的信息来源和逻辑？
- [ ] 我是否忽视了与共识相反的信号？

**黄金分析场景**:
```
❌ 错误示例:
"市场普遍认为黄金会涨，所以我也看涨"
（盲目跟随共识）

✅ 正确做法:
"市场共识看涨（CFTC持仓显示净多头增加）
 但我发现：
 1. ETF持仓实际在下降（与价格背离）
 2. 美元技术面走强
 3. 我的独立判断：短期可能回调"
```

**纠正措施**:
1. 明确写出"市场共识是什么"
2. 寻找与共识相反的数据点
3. 形成独立于共识的观点

---

### 6️⃣ 近期偏差检查 (Recency Bias)

**问题清单**:
- [ ] 我是否过度重视最近的价格走势？
- [ ] 我是否忽视了长期趋势？
- [ ] 我是否因为"最近涨了"而看涨？
- [ ] 我是否检查了历史同期表现？

**黄金分析场景**:
```
❌ 错误示例:
"黄金这周涨了3%，趋势很强，应该继续涨"
（被近期走势影响，忽视长期季节性规律）

✅ 正确做法:
"虽然这周涨了3%，但：
 1. 历史数据显示3-8月是震荡期，单边上涨难以持续
 2. 当前处于震荡期，不宜追涨
 3. 需要等待回调确认支撑"
```

**纠正措施**:
1. 查看更长周期的图表（月线/年线）
2. 对比历史同期表现
3. 关注季节性规律

---

### 7️⃣ 可得性偏差检查 (Availability Heuristic)

**问题清单**:
- [ ] 我是否因为"最近看到"某条新闻而过度重视？
- [ ] 我是否忽视了基础数据，只关注"印象深刻"的信息？
- [ ] 我是否因为某次交易记忆深刻而影响当前判断？

**黄金分析场景**:
```
❌ 错误示例:
"昨天看到新闻说央行大举购金，所以黄金一定会涨"
（可得性偏差，过度重视最近信息）

✅ 正确做法:
"央行购金是长期趋势，但：
 1. 需要查看具体购金量和时间分布
 2. 对比历史购金数据，判断是否超预期
 3. 结合技术面和宏观面综合判断"
```

**纠正措施**:
1. 建立系统化的数据收集流程
2. 使用 checklist 确保不遗漏关键数据
3. 区分"印象深刻"和"重要"

## 🛠️ 偏差纠正工具箱

### 工具1: 反方观点写作

**操作步骤**:
1. 写下你的核心观点（如"看涨黄金"）
2. 强制写出3个反方观点
3. 为每个反方观点寻找数据支持
4. 评估反方观点的合理性

**模板**:
```markdown
我的观点: [看涨/看跌/观望]黄金

反方观点1: [观点]
数据支持: [数据]
合理性评估: [高/中/低]

反方观点2: [观点]
数据支持: [数据]
合理性评估: [高/中/低]

反方观点3: [观点]
数据支持: [数据]
合理性评估: [高/中/低]

结论: [是否修正原观点]
```

### 工具2: 24小时冷静期

**适用场景**: 重大决策前（如大幅加仓/清仓）

**操作步骤**:
1. 产生决策冲动时，先记录下来
2. 等待24小时
3. 24小时后重新评估
4. 如果仍然坚持原决策，再执行

### 工具3: 概率校准

**操作步骤**:
1. 将预测从"点估计"改为"区间估计"
2. 为预测分配置信度
3. 定期回顾预测准确率
4. 调整过度自信/不自信

**示例**:
```
❌ 错误: "下周黄金会涨到$2950"
✅ 正确: "下周黄金有60%概率在$2880-$2950区间，
         突破$2950的概率20%，跌破$2880的概率20%
         置信度: 70%"
```

### 工具4: 决策日志

**记录内容**:
```markdown
## 决策日志

**日期**: [日期]
**决策**: [买入/卖出/持有]
**理由**: [核心逻辑]
**置信度**: [X]%
**潜在风险**: [风险点]
**如果错误，最可能原因**: [原因]

**结果回顾**（一周后填写）:
- 实际结果: [结果]
- 预测准确性: [准确/部分准确/错误]
- 偏差识别: [是否受某种偏差影响]
- 改进措施: [下次如何改进]
```

## 🎯 使用流程

### 每次深度分析前的标准流程

```
1. 打开本检查清单
2. 完成7项偏差检查（约5-10分钟）
3. 填写偏差自检评分表
4. 如整体风险为"高"，执行纠正措施
5. 开始正式分析
6. 分析完成后，记录决策日志
```

### 与深度分析Agent集成

当执行 `/gold-weekly-deep` 时，Agent会自动：
1. 引用本检查清单
2. 在报告中包含"认知偏差自检结果"章节
3. 标注已采取的偏差纠正措施

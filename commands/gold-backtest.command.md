# 定投回测

运行黄金定投策略的历史回测，验证不同策略的表现。

## 支持策略

| 策略 | 说明 | 适用场景 |
|------|------|---------|
| `fixed` | 固定金额定投 | 基础策略，适合新手 |
| `smart` | 智能加仓（跌多买多） | 波动市场，降低成本 |
| `seasonal` | 季节性调仓（小桥流水方法论） | 利用9-10月回调/11-12月布局 |
| `kangbo` | 康波周期配置 | 长期持有，忽略短期波动 |

## 参数说明

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `--strategy` | string | fixed | 定投策略类型 |
| `--start` | string | 2020-01-01 | 回测开始日期 |
| `--end` | string | 今天 | 回测结束日期 |
| `--amount` | number | 1000 | 每次定投金额（元） |

## 使用示例

```bash
# 基础定投回测
/gold-backtest

# 智能加仓策略
/gold-backtest --strategy smart --amount 2000

# 季节性策略（小桥流水方法论）
/gold-backtest --strategy seasonal --start 2019-01-01

# 康波周期策略
/gold-backtest --strategy kangbo --start 2015-01-01
```

## 回测指标

回测报告包含以下指标：

| 指标 | 说明 |
|------|------|
| 总投入 | 累计定投金额 |
| 总份额 | 累计持有份额 |
| 当前市值 | 按最新价格计算的市值 |
| 累计收益 | 市值 - 总投入 |
| 收益率 | 累计收益 / 总投入 |
| 年化收益 | 考虑时间因素的年化收益率 |
| 最大回撤 | 期间最大亏损幅度 |
| 夏普比率 | 风险调整后收益 |
| 交易次数 | 实际定投次数 |

## 策略详解

### 固定金额（fixed）
- 每月固定日期投入固定金额
- 简单易懂，适合长期执行

### 智能加仓（smart）
- 基础定投 + 跌幅触发加仓
- 跌幅>3%：加仓50%
- 跌幅>5%：加仓100%
- 适合波动较大的市场

### 季节性调仓（seasonal）
- 基于小桥流水人家方法论
- 9-10月：减仓50%或暂停定投
- 11-12月：加仓100%
- 其他月份：正常定投

### 康波周期（kangbo）
- 康波萧条期（2015-2025）持续配置
- 忽略短期波动，长期持有
- 适合5年以上投资期限

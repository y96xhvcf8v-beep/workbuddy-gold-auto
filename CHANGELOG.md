# Changelog

All notable changes to this project will be documented in this file.

## [4.2.0] - 2026-03-18

### Added
- 新增行为金融学认知偏差检查清单（7种常见偏差识别与纠正）
- 新增七维置信度量化评估体系
- 新增置信度评估标准（90-100%/80-89%/70-79%/60-69%/<60%）
- 新增反方观点分析要求（每个维度必须包含"如果分析错误，最可能原因"）
- 新增策略切换条件说明
- 新增认知风险类别（MECE风险识别）
- 新增分析质量评估（数据完整性、逻辑严密性等）

### Enhanced
- 优化 Deep Research 阶段：新增认知偏差预检清单
- 优化 Self-Reflecting Chain：明确置信度评估标准和决策流程
- 优化 Complex Reasoning：新增七维置信度汇总表
- 优化综合报告模板：包含认知偏差自检结果、置信度瓶颈分析

### Documentation
- 新增认知偏差检查清单文档（cognitive-bias-checklist.md）
- 更新 README.md 特性说明
- 添加 .codebuddy-plugin/plugin.json 以支持 CodeBuddy 插件市场
